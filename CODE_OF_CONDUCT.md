# Code of Conduct

BlueLearn adopts the **Contributor Covenant v2.1** in full. Read it here:

→ https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies to all project spaces — the GitHub
organization, issues, pull requests, the Discord server, project events,
and any time someone is representing BlueLearn in public.

## Reporting

If you experience or witness behavior that violates the Covenant, please
report it privately to:

**conduct@bluelearn.org**

Reports are received by the project leads and are kept confidential.
Reporters will not be retaliated against.

We will respond within 7 days with next steps. Enforcement decisions
follow the Contributor Covenant's enforcement ladder (correction →
warning → temporary ban → permanent ban) at the discretion of the
project leads.

## Why we adopt the Covenant

It's the most widely-recognized standard in open source. Adopting it
unmodified means new contributors don't have to learn a bespoke set of
rules — they already know what to expect.
